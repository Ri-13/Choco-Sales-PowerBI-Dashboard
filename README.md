🍫 Awesome Chocolates Sales Dashboard in Power BI
Project Overview
An interactive Power BI dashboard built to analyze the sales performance of a fictional chocolate company — Awesome Chocolates.
This project was built as part of my Power BI learning journey, following Chandoo's YouTube tutorial.

📊 KPIs Tracked
- Total Sales
- Total Boxes
- Total Shipments
- Total Costs
- Total Profit

## ✨ Features
- Filter all KPIs by **Country** and **Chocolate Type**
- Track individual **Salesperson** performance
- Check whether **Products** hit or missed their targets
- Month-on-month trend analysis

🛠️ Tools Used
- Power BI Desktop
- Power Query
- DAX (Data Analysis Expressions)
